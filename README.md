### Featured Projects

#### 📡 77 GHz FMCW Radar Simulator - MATLAB *(In Progress)*
Developing a zero-dependency FMCW radar simulator covering PLL-based chirp synthesis, full Tx/Rx chain modeling, 2D FFT range-Doppler processing, and CA-CFAR detection.
→ [fmcw-radar-matlab](https://github.com/IamOumarIbrahim/fmcw-radar-matlab)

#### 🔊 Adaptive Noise Cancellation (NLMS) - Speech Enhancement - MATLAB
Designed and implemented an NLMS adaptive filtering system for real-time speech enhancement under non-stationary noise conditions. Achieved +5.14 dB SNR improvement (2.00 dB → 7.14 dB) and benchmarked against RLS, Kalman-LMS, wavelet-domain, and DNN baselines.
→ [adaptive-noise-cancellation-dsp](https://github.com/IamOumarIbrahim/adaptive-noise-cancellation-dsp)

#### 📻 AM Radio Modulation and Demodulation - MATLAB
Built a complete AM transmission and coherent demodulation chain for recorded voice signals. Applied a 5th-order Butterworth low-pass filter and validated signal recovery under AWGN at 10 dB and 20 dB SNR.
→ [am-modulation-demodulation-matlab](https://github.com/IamOumarIbrahim/am-modulation-demodulation-matlab)

#### 🥤 6-Choice Vending Machine — Full TTL Digital Logic - CircuitMaker
Engineered a fully functional, microcontroller-free vending machine using 50+ discrete TTL ICs. Implemented D flip-flop stock storage, 8-bit ripple-carry adder/subtractor change arithmetic, real-time BCD-to-7-segment decoders, and FSM-based conditional discount logic.
→ [ttl-vending-machine](https://github.com/IamOumarIbrahim/ttl-vending-machine)

#### ☀️ Grid-Connected Solar PV System Design - System Advisor Model (SAM)
Simulated a 367.85 kW DC roof-mounted PV system using Trina Solar modules and SMA inverters. Optimized azimuth and tilt to achieve 629,127 kWh annual AC output with minimal inverter clipping loss.
